创建 src/router/routes.ts 定义路由配置：

定义路由类型接口:
```typescript
export interface RouteConfig {
  path: string
  name: string
  component?: Component
  meta?: {
    title?: string
    requiresAuth?: boolean
    permission?: string | string[]
    layout?: 'default' | 'blank'
  }
  children?: RouteConfig[]
}
```

定义路由列表（使用 D-06: 混合加载策略）:

1. 登录页路由 - 懒加载
   - path: /login
   - name: Login
   - component: () => import('@/views/login/index.vue')
   - meta.layout: 'blank'

2. 主布局路由 - 预加载
   - path: /
   - component: 预加载 DefaultLayout
   - redirect: /dashboard
   - children:
     - /dashboard - 仪表盘
     - /user - 用户管理（Phase 2）
     - /role - 角色管理（Phase 2）
     - /settings - 系统设置（Phase 3）

创建 src/router/guards.ts 实现路由守卫:

1. beforeEach 守卫:
   - 路由元信息中标记 requiresAuth: true 的路由需要登录
   - 未登录用户访问受保护路由 → 重定向到 /login（AUTH-05）
   - 已登录用户访问 /login → 重定向到首页
   - 从 localStorage 读取 token 验证登录状态（per D-01, D-02）

2. afterEach 守卫:
   - 更新页面标题（document.title）

创建 src/router/index.ts 导出路由实例:
- 使用 createRouter 创建路由
- 配置 history 模式（createWebHistory）
- 导出路由实例供 main.ts 使用
